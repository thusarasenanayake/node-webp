### ABOUT

node-webp is a command-line utility that allows you to convert and minimize images into the WebP format.

### SCREENSHOTS

![Screenshot](./assets/Screenshot.png)

### INSTALLATION

```sh
# to install globally
npm install -g .

# to create a symlink in the global package folder
npm link

```
